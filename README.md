Python Socket Chat

Quick Start:

New user, while connecting to the server is requested to create a new identifier. Type 'identifier' -> enter new desired identifier and an identifier unique password.

Existing user, while reconnecting to the server can access their identifier through entering the identifier and password, after typing 'identifier' in the terminal.

To view all online users -> type 'online users list'.

To message one of the online users -> type 'message' and enter the recipient user identifier, then enter the desired text message.

To terminate the user session -> type 'terminate'.

In order to prevent GetPassWarning -> modify run configuration in your IDE to 'Emulate terminal in output console'.
